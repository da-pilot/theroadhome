# Intent

Entry point for the crisis audience. One page that explains how The Road Home helps a person move from unsheltered → housed, and routes them to the right program in one click.

# Audience

**Person in crisis** (primary) — needs shelter tonight or help within the week. Low reading load; expects phone numbers and clear pathways.
**Caseworker / partner referrer** (secondary) — looking up which program to refer to.

# Sections

1. Global nav (home-b grammar)
2. Pillar hero — breadcrumbs + one-line pillar statement + "how does it help?" subline
3. Triage band (phones forward) — same three-lane layout as home
4. Journey triptych — Emergency shelter → Supportive services → Housing assistance
5. Resource-center locations — two cards (Men's Resource Center, Midvale Family Resource Center)
6. Related help (community referrals, landlord resources, housing navigation)
7. Footer
